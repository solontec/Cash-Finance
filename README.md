# Cash&Finance


Projeto desenvolvido para estudos e inserção no mercado real, aplicando conceitos fundamentais em um sistema real ligado ao **mercado financeiro**.

O Cash&Finance consome dados da **Bolsa de Valores em tempo real** por meio de uma API, processando e organizando essas informações para o usuário ter um direcionamento
completamente individual e com base no seu perfi, com recomendações de investimenos, noticias e insights.

## Objetivo
- Aprender a stack (Java + Spring)
- Consumir APIs REST
- Trabalhar com JSON e lógica de negócio
